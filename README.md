# Qingyang Frank Gu's Portfolio using React.js and Chakra UI, Deployed with Github Pages

### Live Deplpoyment: frankqgu.github.io/

### Features:
- News Section
- Header
- Nav Bar
- Footer
- About me
- Projects
- Experiences
- Typography
- Light/Dark dynamic

## Getting Started


```
# Install dependencies
yarn install

# Start the development server
yarn start

# Build the static files
yarn build
```
